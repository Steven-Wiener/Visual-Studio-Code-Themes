## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#173A2E` | Editor canvas |
| Keywords | `#ee255a` | Language keywords, control flow |
| Functions | `#0C92BB` | Function names, method calls |
| Strings | `#55ed79` | String literals |
| Variables | `#0C92BB` | Variables, parameters |
| Constants | `#653BD8` | Numeric literals, constants |
| Comments | `#656565` | Code comments |
| Accent | `#ee255a` | UI highlights, cursor, borders |
