## Color Palette

| Role | Hex | Usage |
|------|-----|-------|
| Background | `#0d0935` | Editor canvas |
| Keywords | `#b700ff` | Language keywords, control flow |
| Functions | `#76ee00` | Function names, method calls |
| Strings | `#f5d300` | String literals |
| Variables | `#08f7fe` | Variables, parameters |
| Constants | `#f5d300` | Numeric literals, constants |
| Comments | `#65375a` | Code comments |
| Accent | `#b700ff` | UI highlights, cursor, borders |

**Source palette:** #05031a · #0d0935 · #191266 · #b700ff · #ff0077 · #08f7fe · #76ee00 · #fe53bb · #f5d300 · #58c4c4
